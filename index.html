A Little About Me

You know that feeling when you're using a well made application? It has the right features, works well, easily does what you want it to and looks great? I love that. I love quality software. Geeking out about a sweet new Android or web app is a common occurrence around my house. The kind of stuff Google, Apple or Basecamp put out. That's what makes the web awesome in my mind. There are so many great applications and software tools available these days. I want to be part of the industry that makes that happen.

Right now, I'm learning as much as I can about web development. I spent a few months learning the basics of front-end development. I enjoyed it quite a bit, but I feel like the back-end is where I really want to be. Specifically, I'm learning how to use Ruby and Ruby on Rails to build killer web apps.

I sincerely love the web. I love the power, connectivity and information it provides. I would love nothing more than to start working for a high quality web development firm building awesome applications and sites. The web is already an amazing place. I genuinely want to be a part of making it better.

When I'm not coding, I enjoy hanging out with my wife and being outdoors.
